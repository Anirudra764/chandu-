# Public Assets

Place your custom profile image in this directory:
- Recommended name: `profile.jpg` (or `profile.png`, `profile.jpeg`, `profile.webp`)

The portfolio is coded to automatically detect and load `/profile.jpg`, `/profile.png`, `/profile.jpeg`, or `/profile.webp` from this directory. If no image is uploaded here, it gracefully falls back to the default Unsplash portrait.
